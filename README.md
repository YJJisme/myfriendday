# 邊緣人鑑定驚喜網頁 (B-Index Surprise)

這是一個偽裝成「邊緣人鑑定測驗」的生日驚喜網頁。透過一連串的心理測驗與「虛假的等待分析」，在特定時刻（2026年2月5日）為壽星揭曉真正的生日祝福。

## 1. 核心設計規範

*   **視覺風格**：極簡黑底 (`#000000`) 搭配白字 (`#FFFFFF`)，字體選用 Google Fonts `Courier Prime`，營造冷靜、科技感的分析介面。
*   **安全防護**：全站禁用右鍵、選取反白與 Ctrl+S 存檔功能。
*   **裝置限制**：強制橫向螢幕瀏覽，直立時會顯示遮罩。

## 2. 專案邏輯架構 (Workflow)

```mermaid
graph TD
    A[進入網頁] --> B{檢查時間 > 2026/02/05?}
    B -- Yes --> F[顯示鑑定結果 (Result Phase)]
    B -- No --> C{檢查 localStorage quiz_completed?}
    C -- Yes --> E[顯示分析等待畫面 (Analysis Phase)]
    C -- No --> D[顯示測驗題目 (Quiz Phase)]
    D --> |提交分析| E
    E --> |時間到達| F
    F --> |查看報告| G[揭曉信封 (Reveal Phase)]
    G --> |點擊信封| H[打字機祝福語]
```

### 階段說明
1.  **測驗階段 (The Quiz)**：
    *   3 題關於社交習慣的選擇題。
    *   提交後會將狀態寫入瀏覽器 `localStorage`，確保重新整理後不會重複測驗。
2.  **分析等待 (The Wait)**：
    *   進度條會快速跑至 **99.9%** 並停住，營造「精密運算中」的假象。
    *   顯示倒數計時器，精確到秒。
    *   提供 `.ics` 日曆下載功能，讓使用者設定提醒。
3.  **結果揭曉 (The Result)**：
    *   時間一到（或測試模式下），進度條補滿 100%。
    *   顯示「邊緣人指數：0%」的反轉結果。
4.  **驚喜信封 (The Reveal)**：
    *   點擊查看詳情後，信封浮現。
    *   點開信封，顯示核心祝福語。

## 3. 測試模式 (Test Mode)

開發者或預覽時，可使用 URL 參數強制跳過等待時間，直接檢視最終結果。

*   **使用方式**：在網址後方加上 `?test=true`
*   **範例**：`index.html?test=true`
*   **效果**：無視目前日期與測驗狀態，直接進入結果揭曉階段。

## 4. 視聽體驗優化 (Enhancements)

為了提升沉浸感，本專案已實作以下優化：

*   **流星特效 (Shooting Stars)**：
    *   背景隨機出現流星動畫，營造浪漫氛圍。
*   **打字機音效 (Typewriter SFX)**：
    *   使用 Web Audio API 即時生成，無需外部音效檔。
    *   在祝福語打字時同步播放機械按鍵聲。
*   **背景音樂 (BGM) 支援**：
    *   已預留 BGM 播放程式碼。
    *   **啟用方式**：請將您的音樂檔案命名為 `bgm.mp3` 並放置於專案根目錄（與 `index.html` 同層級）。
    *   音樂將在使用者點擊「查看詳細分析報告」時自動淡入播放。

## 5. 未來優化接口 (Future Optimization)

*   **更多題目**：
    *   可在 HTML `quiz-phase` 區塊中依照現有結構複製新增題目。
*   **慶生背景圖片**：
    *   兩種方式啟用：
        - 方式 A：在專案根目錄放入 `birthday-bg.jpg`（與 `index.html` 同層級），頁面會自動套用到「揭曉階段」。
        - 方式 B：使用 URL 參數指定圖片，例如：`index.html?bg=your-image.jpg` 或提供完整連結 `index.html?bg=https://example.com/pic.jpg`。
    *   文字仍維持白色，並加上一層半透明遮罩以確保可讀性。
